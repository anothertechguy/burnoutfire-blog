# Competitor Analysis

## Overview

This document tracks competitive analysis for BurnoutFIRE blog content. Analysis is conducted for each pillar article and key supporting content.

## Methodology

1. Identify top 10-20 ranking pages for target keywords
2. Extract content metrics (word count, structure, depth)
3. Identify content gaps and opportunities
4. Benchmark quality metrics
5. Document unique angles and differentiators

## Analysis Schedule

- **Initial**: Conducted for 3 pillar articles
- **Updates**: Monthly for high-performing keywords
- **Deep dives**: Quarterly for competitive keywords
